# Random Food API Demo 🍔

**[🔴 Live Demo Available Here!](https://cyber752.github.io/Random-Food-API-Demo/)**

A lightweight, simple demonstration of how to consume a free public REST API using Vanilla JavaScript, HTML, and CSS. 

## Features
- **Fetch API**: Uses modern `async/await` syntax to retrieve JSON data.
- **TheMealDB API**: Integrates with [TheMealDB](https://www.themealdb.com/api.php) to grab high-quality, random images of food from around the world.
- **Zero Dependencies**: Built entirely with standard web technologies. No React, no Vue, no jQuery, no NPM installs required!

## How to Run
Since this is a static frontend project, you don't need any complex servers to run it.
1. Clone or download this repository.
2. Open the `index.html` file directly in any modern web browser.
3. Click the "Search" button to see it in action!

## Files Included
- `index.html`: The structure and styling (CSS) for the UI.
- `app.js`: The logic that handles the button click, makes the HTTP network request, parses the JSON, and updates the DOM dynamically.

## Learning Objectives
This project was built to understand:
1. Working with APIs and Endpoints.
2. Handling asynchronous data in JavaScript (`try/catch/finally`).
3. Navigating JSON objects.
4. Dynamically updating an HTML `<img>` source based on a network response.
